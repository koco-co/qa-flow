# 图N 页面要点格式

每张图片引用下方统一插入一个结构块，使用以下格式：

```markdown
#### 图N 页面要点

- 页面类型：
- 区域构成：
- 关键操作：
- 列表信息：
- 检索条件：
- 输入项：
- 流程/状态：
- 识别限制：
```

## 填写规则

- `图N` 按图片在文档中的出现顺序连续编号。
- 仅记录图片中**可直接读出**的信息，不杜撰按钮、字段或状态。
- 某项不涉及时写 `无` 或 `不涉及`，不要留含糊描述。
- 图片模糊、遮挡或裁切导致无法确认的信息，统一写入 `识别限制`。

## 歧义处理规则

图片中存在以下情况时，**不得直接填写推测内容**，必须标入 `识别限制` 并向用户提出带选项的确认问题：

- 文字因分辨率/截断而无法完整读出
- 按钮名、字段名有多种可能的读法
- 图中有嵌套截图（如 Excel 表格截图）但内容不清晰
- 流程箭头指向不确定

**确认问题格式：**

```
❓ 图N 中以下内容无法确认，请选择：

**[不确定项]**
  [1] [选项A]
  [2] [选项B]  ← AI 推荐
  [3] 其他（请输入）：___
```

等待用户确认后再补充该内容，**不得提前写入文档**。
