# 数据资产主流程测试用例设计文档

## 概述

基于源码分析，为**标品 v6.3** 和**岚图定制版**数据资产编写主流程回归测试用例（MD 格式），为后续 UI 自动化做准备。

## 输出产物

| 文件 | 路径 | 预估用例数 |
|---|---|---|
| 标品v6.3-主流程用例.md | workspace/archive/数据资产-主流程用例/ | ~120 |
| 岚图-主流程用例.md | workspace/archive/数据资产-主流程用例/ | ~135 |

## 设计原则

- **纯功能测试视角**：用户操作 + UI 验证，不包含路由/API/技术实现
- **源码驱动**：以源码中实际页面结构为准，CSV 参考用例为骨架
- **步骤可执行**：每个步骤明确操作对象和预期结果
- **偏好规则优先**：遵循 preferences/case-writing.md 中的按钮名称、菜单路径等规范
