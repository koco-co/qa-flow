## 数据货架管理 - 数据集管理

### 列表页

#### 页面初始化与权限

##### 【P0】验证列表页默认展示DataWorks数据集页签字段排序与页面级按钮

> 前置条件
```
1) 使用具备【数据货架管理 → 数据集管理】页面访问权限的后台账号登录系统。
2) DataWorks数据集与手工数据集页签下均存在可展示数据。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架管理 → 数据集管理】页面，等待列表加载完成 | 页面默认进入数据集管理列表页；默认高亮「DataWorks数据集」页签 |
| 2 | 在列表区域查看表头、默认排序和页面级按钮 | 列表展示 PRD 定义的字段列与操作列；默认按编目更新时间排序；页面顶部显示【立即同步】按钮 |
| 3 | 在页面顶部切换到「手工数据集」页签，等待列表刷新完成 | 页签切换成功；列表展示手工数据集数据；页面顶部显示【新建数据集】按钮 |

##### 【P1】验证只读账号进入列表页时仅显示详情入口

> 前置条件
```
1) 使用仅具备页面查看权限和详情权限的只读账号登录系统。
2) 列表中存在可查看的数据集记录。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架管理 → 数据集管理】页面，等待列表加载完成 | 页面可正常访问；页面顶部不显示【立即同步】按钮；切换到「手工数据集」页签后不显示【新建数据集】按钮 |
| 2 | 在列表区域查看任一数据行的操作列，点击【详情】按钮，等待详情页打开 | 操作列仅显示【详情】入口；不显示【编目】【发布】【上架】【下架】等操作；可正常进入详情页 |

#### 搜索筛选与分页

##### 【P1】验证按名称状态时间筛选后可重置并在结果集中正确翻页

> 前置条件
```
1) 使用具备查询权限的后台账号登录系统。
2) 列表中存在可满足名称、状态、更新时间、折叠已删除和分页验证的数据。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架管理 → 数据集管理】页面，等待列表加载完成 | 默认展示 DataWorks 数据集列表；列表初始显示默认结果 |
| 2 | 在筛选区输入名称关键字，选择上架状态、更新时间范围和「折叠已删除」条件后点击【查询】按钮，等待列表刷新完成 | 列表按输入条件返回结果；筛选区保留已输入和已选择的条件；结果与条件匹配 |
| 3 | 在筛选区点击【重置】按钮，等待列表刷新完成 | 名称、状态、时间范围和折叠已删除条件恢复默认；列表恢复默认结果 |
| 4 | 再次输入可命中多页数据的关键字后点击【查询】按钮，等待列表刷新完成 | 查询结果支持分页，且已输入的查询条件保持不变 |
| 5 | 在分页器点击下一页，等待列表刷新完成 | 分页切换成功；当前页高亮正确；列表数据随分页变化 |

#### 状态流转

##### 【P0】验证编目不完备且未发布的数据集可直接发布并上架再下架

> 前置条件
```
1) 使用具备发布和上下架权限的后台账号登录系统。
2) 列表中存在一条编目状态为不完备、发布状态为未发布、上架状态为未上架的数据集记录。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架管理 → 数据集管理】页面，等待列表加载完成 | 目标数据集可见，且状态显示为「不完备 / 未发布 / 未上架」 |
| 2 | 在目标数据行点击【发布】按钮并完成确认，等待列表刷新完成 | 目标数据集的发布状态变为「已发布」；页面未出现阻断操作的异常提示 |
| 3 | 在同一数据行点击【上架】按钮并完成确认，等待列表刷新完成 | 目标数据集的上架状态变为「已上架」；操作列显示【下架】按钮 |
| 4 | 在同一数据行点击【下架】按钮并完成确认，等待列表刷新完成 | 目标数据集的上架状态变回「未上架」；发布状态保持为「已发布」 |

#### 前后台联动

##### 【P0】验证后台上架后前台可见且下架后前台不可见

> 前置条件
```
1) 使用具备数据集管理操作权限的后台账号登录系统。
2) 存在一条已完成编目、已发布但尚未上架的手工数据集。
3) 存在可访问【数据货架 → 数据集资源】页面的前台账号。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架管理 → 数据集管理】页面，等待列表加载完成后切换到「手工数据集」页签并定位目标数据集 | 目标数据集可见，且操作列显示【上架】按钮 |
| 2 | 在目标数据行点击【上架】按钮并完成确认，等待列表刷新完成 | 目标数据集的上架状态变为「已上架」；操作列显示【下架】按钮 |
| 3 | 进入【数据货架 → 数据集资源】页面，使用资源名称或关键字搜索该数据集 | 前台可搜索到该资源，并可看到对应资源卡片 |
| 4 | 返回【数据货架管理 → 数据集管理】页面，对同一数据集执行【下架】操作后，再次进入【数据货架 → 数据集资源】页面搜索该资源 | 后台上架状态变为「未上架」；前台搜索结果中不再显示该资源 |

1) 使用具备数据集编目和发布权限的后台账号登录系统。

| 2 | 在目标数据行点击【编目】按钮，修改基础信息中的可编辑内容后点击【保存编目】按钮，等待返回列表页 | 页面提示保存成功；返回列表页后，目标数据集的发布状态变为「待发布」 |
| 3 | 进入【数据货架 → 数据集资源】页面，打开该资源详情并查看被修改的信息 | 前台仍展示上一次已发布版本的内容；本次仅保存未发布的修改尚未生效 |
| 4 | 返回【数据货架管理 → 数据集管理】页面，对目标数据集执行重新发布后，再次进入前台详情页查看 | 目标数据集重新发布成功；前台详情页切换为最新发布版本，展示本次修改后的内容 |

| 2 | 在页面顶部点击【立即同步】按钮并完成确认，等待列表刷新完成 | 同步任务发起成功；页面同步时间更新；DataWorks 数据集信息按同步结果刷新 |
| 3 | 进入【数据货架 → 数据集资源】页面，查看前台资源列表和同步时间 | 前台同步时间同步更新；同步后的 DataWorks 资源可正常展示；已有手工资源未被错误覆盖或替换 |

##### 【P1】验证注册API缺失字段显示为空态且不错误回显向导API或脚本API数据

> 前置条件
```
1) 存在一条已发布且已上架的注册 API 资源。
2) 该资源部分技术元数据字段未维护，用于验证空态展示。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架 → API资源】页面，等待卡片区加载完成 | 页面可展示目标注册 API 资源卡片 |
| 2 | 点击目标 API 卡片，等待详情页打开 | 详情页成功打开 |

##### 【P0】验证无解锁权限时API调用地址保持锁定且不可复制

> 前置条件
```
1) 存在一条已发布且已上架、并已维护 API 调用地址的 API 资源。
2) 使用具备详情查看权限但不具备「解锁技术元数据 API 调用地址」权限的前台账号登录系统。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架 → API资源】页面，等待卡片区加载完成 | 页面可展示目标 API 资源卡片 |
| 2 | 点击目标 API 卡片，等待详情页打开 | 详情页成功打开 |
| 3 | 点击【技术元数据】Tab，等待内容区加载完成 | 「API调用地址」字段保持锁定状态；页面不显示【解锁】入口；不展示完整地址 |
| 4 | 点击锁定状态下的「API调用地址」字段区域 | 地址仍保持锁定状态；页面不展示完整地址；不可执行复制完整地址的操作 |

##### 【P0】验证有解锁权限时可解锁并复制完整API调用地址且返回上层保留筛选条件

> 前置条件
```
1) 存在一条已发布且已上架、并已维护 API 调用地址的 API 资源。
2) 使用具备「解锁技术元数据 API 调用地址」权限的前台账号登录系统。
3) API 资源列表页支持搜索条件保留。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ------------ | ------------ |
| 1 | 进入【数据货架 → API资源】页面，在搜索栏输入关键字后点击【搜索】按钮，等待卡片区刷新完成 | 搜索条件生效；结果列表展示目标 API；搜索框保留输入内容 |
| 2 | 点击目标 API 卡片，等待详情页打开 | 详情页成功打开；头部统计信息展示正常 |
| 3 | 点击【技术元数据】Tab后，点击「API调用地址」字段旁的【解锁】按钮，等待地址展示区域刷新完成 | 地址由锁定状态切换为可见状态；页面展示完整调用地址 |
| 4 | 点击已解锁的「API调用地址」字段或复制入口 | 系统可执行复制操作，并提供明确的成功反馈；复制内容为完整地址 |
| 5 | 在详情页完成收藏切换后点击【返回上层】按钮，等待资源列表页重新加载完成 | 收藏操作生效；返回列表页后，原搜索条件和结果列表保持不变 |
