# 【内置规则丰富】有效性，支持设置字段多规则的且或关系

## 需求概述

| 项目     | 内容                                                               |
| -------- | ------------------------------------------------------------------ |
| 开发版本 | 6.3岚图定制化分支（数据资产V6.4.10） 🟢                            |
| 需求背景 | 有效性校验支持校验取值范围和枚举值的合并判断，支持配置且/或关系 🟢 |
| 影响模块 | 数据质量 → 规则配置、结果查询、规则库 🟢                           |
| 导航路径 | 数据质量 → 数据校验 → 规则配置 / 结果查询 🔵 `/dq/rule`            |
| 关联需求 | 独立需求，不依赖其他 #15693/#15694/#15696 🟢                       |

## 页面：规则配置 - 新增"取值范围&枚举范围"联合规则

> 📍 导航路径：数据质量 → 数据校验 → 规则配置 → 添加/编辑规则
> 📸 来源：蓝湖 #15695

![过滤条件](images/1-u109.png)

### 规则基本信息

| 字段名   | 值                | 来源 |
| -------- | ----------------- | ---- |
| 规则类型 | 字段级            | 🟢   |
| 规则名称 | 取值范围&枚举范围 | 🟢   |
| 规则分类 | 有效性校验        | 🟢   |
| 关联范围 | 字段              | 🟢   |

### 字段定义 — 取值范围&枚举范围配置

| 字段名                | 控件类型       | 必填 | 校验规则                       | 默认值 | 来源 |
| --------------------- | -------------- | ---- | ------------------------------ | ------ | ---- |
| 字段                  | 下拉选择(单选) | 是   | 选择待校验的字段               | -      | 🟢   |
| 取值范围-运算符       | 下拉选择       | 是   | 支持 >=, <=, >, < 等           | -      | 🟢   |
| 取值范围-期望值       | 输入框         | 是   | 数值类型                       | -      | 🟢   |
| 枚举值设置-包含关系   | 下拉选择       | 是   | in / not in                    | -      | 🟢   |
| 枚举值设置-枚举值列表 | 输入区域       | 是   | 逗号分隔的值列表               | -      | 🟢   |
| 取值范围和枚举值关系  | 单选（且/或）  | 是   | 配置取值范围和枚举值的逻辑关系 | 且     | 🟢   |
| 过滤条件              | 选项配置+按钮  | 否   | 点击按钮进行配置               | -      | 🟢   |
| 强弱规则              | 单选           | 是   | 强规则/弱规则                  | 强规则 | 🟢   |
| 规则描述              | 文本输入       | 否   | 自定义描述                     | -      | 🟢   |

### 交互逻辑

1. 新增"取值范围&枚举范围"联合规则选项，位于规则类型选择中 🟢
2. 第一行：取值范围设置，逻辑和原有取值范围规则保持一致（运算符 + 期望值，支持多行条件） 🟢
3. 第二行：枚举值设置，逻辑和枚举值规则保持一致，支持选择 in/not in 🟢
4. 第三行：取值范围和枚举值的且/或关系配置 + 过滤条件 🟢
5. 新增的规则支持 string、数值类型字段判断，string 类型默认强转为 double 类型 🟢
6. **注意**：原来的枚举值配置规则也同步新增 in/not in 选项（影响已有枚举值规则） 🟢

### 异常处理

| 场景                            | 系统行为           | 来源 |
| ------------------------------- | ------------------ | ---- |
| string 类型字段强转 double 失败 | 校验失败，记录异常 | 🔴   |
| 取值范围期望值为非数值          | 输入校验不通过     | 🔴   |
| 枚举值列表为空                  | 必填校验提示       | 🟢   |

## 页面：结果查询 - 质量报告展示

> 📍 导航路径：数据质量 → 数据校验 → 结果查询
> 📸 来源：蓝湖 #15695

### 质量报告列表字段

| 列名       | 说明                               | 来源 |
| ---------- | ---------------------------------- | ---- |
| 规则类型   | 有效性校验                         | 🟢   |
| 规则名称   | 取值范围&枚举范围 / 枚举值         | 🟢   |
| 字段名称   | 校验字段名                         | 🟢   |
| 字段类型   | json / string / 数值等             | 🟢   |
| 质检结果   | 校验通过 / 校验不通过              | 🟢   |
| 未通过原因 | 校验通过时为"--"；不通过时展示原因 | 🟢   |
| 详情说明   | 描述具体规则匹配情况               | 🟢   |
| 操作       | 校验不通过时可"查看详情"           | 🟢   |

### 详情说明文案规则

**取值范围&枚举范围联合规则：**

| 质检结果   | 详情说明文案                               | 来源 |
| ---------- | ------------------------------------------ | ---- |
| 校验通过   | 符合规则"取值范围>1"且"枚举值in '1,2,3'"   | 🟢   |
| 校验不通过 | 不符合规则"取值范围>1"且"枚举值in '1,2,3'" | 🟢   |

> 文案中"且"/"或"根据配置的关系动态替换

**枚举值规则（已有规则变更）：**

| 质检结果   | 详情说明文案                                                                               | 来源 |
| ---------- | ------------------------------------------------------------------------------------------ | ---- |
| 校验通过   | 字段枚举值不存在约定范围外的值，符合规则"枚举值in '1,2,3'"                                 | 🟢   |
| 校验不通过 | 字段枚举值存在约定范围外的值，约定范围外的值的数量总计为xx个，不符合规则"枚举值in '1,2,3'" | 🟢   |

> 枚举值的质量报告详情说明增加不包含(not in)的场景描述 🟢

### 交互逻辑

1. 校验通过的规则，不记录明细数据，操作列为"--" 🟢
2. 校验不通过的规则，操作列展示"查看详情"链接 🟢
3. 校验失败（执行异常）的规则，支持查看日志 🟢
4. 未通过原因：联合规则显示"不符合有效性规则"；枚举值规则显示"枚举值校验未通过" 🟢
