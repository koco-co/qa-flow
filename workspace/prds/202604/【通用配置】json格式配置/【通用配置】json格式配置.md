## 页面：新增/编辑 key 弹窗

> 📍 触发：点击列表页"新增key"按钮 或 "编辑"操作
> 📸 来源：蓝湖 #15696

![页面元素-导入弹窗](images/5-u182.png)

### 字段定义

| 字段名     | 控件类型       | 必填 | 校验规则                              | 默认值         | 来源 |
| ---------- | -------------- | ---- | ------------------------------------- | -------------- | ---- |
| key        | 输入框         | 是   | 最大255字符，无需重复性校验           | 空             | 🟢   |
| 中文名称   | 输入框         | 否   | 最大255字符                           | 空             | 🟢   |
| value格式  | 输入框         | 否   | 最大255字符，输入正则表达式           | 空             | 🟢   |
| 数据源类型 | 下拉选择(单选) | 是   | sparkthrift2.x/hive2.x/doris3.x       | sparkthrift2.x | 🟢   |
| 测试数据   | 输入框         | 否   | 当value格式有值时才显示，用于正则测试 | 空             | 🟢   |

### 交互逻辑

1. 点击"新增key"打开新增弹窗，填写完成后点击"确定"保存 🟢
2. 点击"编辑"打开编辑弹窗，回显当前层级的key名称、中文名称、value格式、数据源类型信息 🟢
3. 数据源类型为必填单选项，默认选中sparkthrift2.x 🟢
4. key无需进行重复性校验 🟢
5. 支持对输入的正则内容进行测试：当value格式配置了才会出现测试数据字段，输入测试数据后可验证正则是否匹配 🟢
6. 点击"取消"关闭弹窗，不保存数据 🟢

### 异常处理

| 场景                          | 系统行为                             | 来源 |
| ----------------------------- | ------------------------------------ | ---- |
| key为空提交                   | 必填校验提示                         | 🟢   |
| key超过255字符                | 字符超限提示                         | 🟢   |
| 中文名称/value格式超过255字符 | 字符超限提示                         | 🟢   |
| 数据源类型未选择              | 必填校验提示（默认有值，理论不触发） | 🟢   |

> 📍 触发：点击列表页"导入"按钮

![导入弹窗](images/5-u182.png)
![导入模板-一层](images/7-u259.png)

### 导入弹窗字段

| 字段名       | 控件类型   | 必填 | 说明                               | 来源           |
| ------------ | ---------- | ---- | ---------------------------------- | -------------- |
| 重复处理规则 | 单选按钮组 | 是   | 重复则跳过（默认）/ 重复则覆盖更新 | 🔵（截图识别） |
| 上传文件     | 文件上传   | 是   | 支持XLSX文件类型，可下载模板       | 🟢             |

### 导入模板结构

| Sheet名 | 字段A（必填）     | 字段B（必填） | 字段C     | 字段D     |
| ------- | ----------------- | ------------- | --------- | --------- |
| 一层    | \*key             | 中文名称      | value格式 | -         |
| 二层    | \*上一层级的key名 | \*key         | 中文名称  | value格式 |
| 三层    | \*上一层级的key名 | \*key         | 中文名称  | value格式 |
| 四层    | \*上一层级的key名 | \*key         | 中文名称  | value格式 |
| 五层    | \*上一层级的key名 | \*key         | 中文名称  | value格式 |

> 🔵 一层 key 列批注："必填，不可超过255字符"
> 🔵 二层起"上一层级的key名"批注："注意若上一层级存在重复的key名，默认取第一个key作为上一层级"

### 交互逻辑

1. 点击"导入"打开导入弹窗 🟢
2. 支持下载导入模板（json_format_import_template.xlsx） 🟢
3. 导入数据均为新增逻辑 🟢
4. 点击确认后进行内容校验 🟢
5. 校验规则：🟢
   - key名填写名称过长（超255字符）→ 标红，批注错误原因
   - 必填项未填写 → 标红，批注"必填项未填写"
   - 上一层级key无法找到 → 标红，批注错误原因
6. 存在错误时弹出提示"导入表格中存在错误数据，请检查后重新导入，点击导出错误文件" 🟢
7. 错误文件命名为"json*format_error_20240520.xlsx"（json_format_error*+日期） 🟢
8. 全部数据均正确后才可以进行导入 🟢
9. 重复处理规则：重复则跳过 / 重复则覆盖更新 🔵（截图识别）

### 异常处理

| 场景                | 系统行为                            | 来源 |
| ------------------- | ----------------------------------- | ---- |
| 上传非xlsx文件      | 文件类型校验失败提示                | 🔴   |
| 导入数据含错误      | 弹出提示，支持导出错误文件          | 🟢   |
| 上一层级key不存在   | 该行标红，批注"上一层级key无法找到" | 🟢   |
| key超过255字符      | 该行标红，批注"key名填写名称过长"   | 🟢   |
| 必填项（key）未填写 | 该行标红，批注"必填项未填写"        | 🟢   |
