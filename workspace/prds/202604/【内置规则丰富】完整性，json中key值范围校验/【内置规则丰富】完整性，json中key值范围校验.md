# 【内置规则丰富】完整性，json中key值范围校验

## 需求概述

| 项目     | 内容                                                          |
| -------- | ------------------------------------------------------------- |
| 开发版本 | 6.3岚图定制化分支（数据资产V6.4.10） 🟢                       |
| 需求背景 | 完整性校验支持对json类型的字段做key值范围的校验 🟢            |
| 影响模块 | 数据质量 → 规则配置、结果查询、规则库 🟢                      |
| 导航路径 | 数据质量 → 数据校验 → 规则配置 / 结果查询 🔵 `/dq/rule`       |
| 关联需求 | 依赖 #15696【通用配置】json格式配置（提供key选择列表数据） 🟢 |

## 页面：规则配置 - 新增"key范围校验"规则

> 📍 导航路径：数据质量 → 数据校验 → 规则配置 → 添加/编辑规则
> 📸 来源：蓝湖 #15693

### 规则基本信息

| 字段名   | 值          | 来源 |
| -------- | ----------- | ---- |
| 规则类型 | 字段级      | 🟢   |
| 规则名称 | key范围校验 | 🟢   |
| 规则分类 | 完整性校验  | 🟢   |
| 关联范围 | 字段        | 🟢   |

### 字段定义 — key范围校验配置

| 字段名   | 控件类型       | 必填 | 校验规则                                         | 默认值 | 来源 |
| -------- | -------------- | ---- | ------------------------------------------------ | ------ | ---- |
| 字段     | 下拉选择(单选) | 是   | 当选择key范围校验时，字段仅支持单选              | -      | 🟢   |
| 统计函数 | 下拉选择       | 是   | 选择"key范围校验"                                | -      | 🟢   |
| 校验方法 | 下拉选择       | 是   | 包含/不包含                                      | 包含   | 🟢   |
| 校验内容 | 多选下拉(树形) | 是   | 选择列表为#15696维护的key数据，支持多选/选择全部 | -      | 🟢   |
| 过滤条件 | 选项配置+按钮  | 否   | 点击按钮进行配置                                 | -      | 🟢   |
| 强弱规则 | 单选           | 是   | 强规则/弱规则                                    | 强规则 | 🟢   |
| 规则描述 | 文本输入       | 否   | 自定义描述                                       | -      | 🟢   |

### 校验内容选择器（依赖#15696）

| 特性         | 说明                                                      | 来源 |
| ------------ | --------------------------------------------------------- | ---- |
| 数据来源     | "json格式配置"(#15696)维护的数据列表，取key名             | 🟢   |
| 支持多选     | 可多选key，支持"选择全部"                                 | 🟢   |
| 选择框内搜索 | 支持输入查询                                              | 🟢   |
| 层级展示     | 按层级展示key结构（如key1→key2为父子层级）                | 🟢   |
| 默认加载     | 若key数据量多，默认加载前200条展示                        | 🟢   |
| 勾选范围     | 勾选仅对当前层级生效                                      | 🟢   |
| 回显格式     | "key1-key2;key11-key22"（多层级用"-"连接，多组用";"分隔） | 🟢   |
| 悬浮提示     | 鼠标悬浮展示全部key名信息，默认仅展示前两个               | 🟢   |
| 支持数据源   | doris3.x、sparkthrift2.x、hive2.x                         | 🟢   |
| 支持字段类型 | key范围校验仅支持数据类型为json、string的字段             | 🟢   |

### 交互逻辑

1. 在统计函数下拉中选择"key范围校验"，悬浮提示："当选择key范围校验时，字段仅支持单选" 🟢
2. 悬浮提示内容"校验内容key信息需要在通用配置模块维护。" 🟢
3. 校验方法支持选择包含/不包含 🟢
4. 配置后需要按照层级进行校验，key名需要按照层级匹配是否存在key信息 🟢
5. 需要考虑key数量几千个的情况（性能要求） 🟢
6. key范围校验仅支持数据类型为json、string的字段 🟢
7. 支持doris3.x、sparkthrift2.x、hive2.x数据源 🟢

### 异常处理

| 场景                    | 系统行为              | 来源 |
| ----------------------- | --------------------- | ---- |
| 字段类型非json/string   | 不展示key范围校验选项 | 🟢   |
| #15696中无key数据       | 校验内容选择框为空    | 🔴   |
| key数据量超大（几千个） | 默认加载前200条       | 🟢   |

## 页面：结果查询 - 质量报告展示

> 📍 导航路径：数据质量 → 数据校验 → 结果查询

### 质量报告列表字段

| 列名       | 说明                                        | 来源 |
| ---------- | ------------------------------------------- | ---- |
| 规则类型   | 完整性校验                                  | 🟢   |
| 规则名称   | key范围校验                                 | 🟢   |
| 字段名称   | 校验字段名                                  | 🟢   |
| 字段类型   | json                                        | 🟢   |
| 质检结果   | 校验通过 / 校验不通过                       | 🟢   |
| 未通过原因 | 校验通过时"--"；不通过时"key范围校验未通过" | 🟢   |
| 详情说明   | 描述具体规则匹配情况                        | 🟢   |
| 操作       | 校验不通过时可"查看详情"                    | 🟢   |

### 详情说明文案规则

| 质检结果   | 详情说明文案                                          | 来源 |
| ---------- | ----------------------------------------------------- | ---- |
| 校验通过   | 符合规则key范围包含/不包含"key1-key2;key11-key22……"   | 🟢   |
| 校验不通过 | 不符合规则key范围包含/不包含"key1-key2;key11-key22……" | 🟢   |

### 交互逻辑

1. 校验通过的规则，不记录明细数据 🟢
2. 校验不通过的规则，支持"查看详情" 🟢
3. 校验失败的规则，支持查看日志 🟢
